Abstract

Applying multiple themes to a sample plot: number_project vs average_montly_hours

Default theme

Code 
p <- ggplot(data = df, mapping = aes(x = factor(number_project), y = average_montly_hours, colour = salary))
p + geom_boxplot()

ggplot2 alternatives

ggplot2 provides the following set of alternative themes:

  • theme_grey()
  • theme_gray()
  • theme_bw()
  • theme_linedraw()
  • theme_light()
  • theme_dark()
  • theme_minimal()
  • theme_classic()
  • theme_void()
  • theme_test()

ggplot2 themes can be applied ad hoc, like in Figure 1, or using the theme_set() function, which changes the default theme.

Code 
p + geom_boxplot() + theme_minimal()

Experiment with ggplot2 themes to discover the theme that you want to pass to theme_test().

ggthemes alternatives

The ggplot2 community has also created some popoular extensions providing ‘extra’ themes. ggthemes is one of these extensions. This time we change the default theme to theme_economist(). As Figure 2 shows, the colors of the visual forms have not changed though (they are ggplot2’s default). To adopt a theme and its colors, we have to pass a further option, as per Figure 3

Code 
theme_set(theme_economist())
p + geom_boxplot()
Code 
p + geom_boxplot() + scale_color_economist()

hrbrthemes alternatives

hrbrthemes is yet another extension offering extra themes for ggplot2. A distinctive feature of this extension is the selection of fonts associatied with the various themes. For example, the theme theme_ipsum comes with seven alternative fonts.

Code 
theme_set(theme_ipsum())
p + geom_boxplot()
Code 
theme_set(theme_ipsum_rc())
p + geom_boxplot()

hrbrthemes offers ‘dark’ themes too (see Figure 6)

Code 
theme_set(theme_ft_rc())
p + geom_boxplot() + scale_color_ft()
